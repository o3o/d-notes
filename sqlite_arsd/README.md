Esempio di connessione a sqlite

Vedere [wiki](https://github.com/o3o/d-notes/wiki/SQLite-drivers)

# Struttura
```
sqlite_arsd/
├── common.mk
├── db
│   └── ldg.sqlite
├── lib
│   └── sqlite3.dll
├── makefile
├── src
│   ├── app.d
│   └── arsd
│       ├── database.d
│       └── sqlite.d
└── update.sh
```
* `src/arsd` sorgenti
* `update` copia i sorgenti 
* `db` database di appoggio
